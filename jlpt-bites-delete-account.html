JLPT Bites – Account Deletion Request

If you would like to delete your JLPT Bites account and all associated data, you can submit a deletion request using the form below.

What will be deleted?

  • Your Firebase Authentication account (Google Sign-In / email identity).
  • Any profile information stored in Firebase Authentication.

What will NOT be deleted?

  • Analytics data stored in aggregate (non-identifiable) form.
  • Crash logs that do not contain personal identifiers.

How it works

  1. You submit the request using the button below.
  2. Your request is reviewed manually by the developer.
  3. Your Firebase Authentication account is permanently deleted within 3–7 days.
